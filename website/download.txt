<h2>IMP 1.0 stable release</h2>

<p>This release is recommended for beginners. It does, however, lack some of
the newer functionality available in the <a href="#nightly">nightly
builds</a>.</p>

<h3>Signatures</h3>

<p>All of the RPMs are signed with
<a href="http://salilab.org/~ben/pubkey.asc">this GPG key</a>.
For all other packages, please check the
<a href="http://salilab.org/imp/1.0/download/md5sum">MD5SUM</a> and
<a href="http://salilab.org/imp/1.0/download/md5sum.asc">its signature</a>.
</p>

<a name="nightly"></a>
<h2>Nightly builds</h2>

<p>Nightly builds are not as well tested as the
<a href="#stable">stable releases</a> but do include new functionality
(they are updated daily).</p>

<h3>Binary installers</h3>

<p>Installers for Windows, Mac OS X 10.4 (Tiger) and Mac OS X 10.6 (Snow
Leopard) <a href="http://salilab.org/imp/nightly/download/">are
available</a>.</p>

<h3>Documentation</h3>

<p>Please refer to the
<a href="http://salilab.org/imp/nightly/doc/html/">online documentation</a>.
</p>

<h3>Source code</h3>

<p>Latest development IMP source code can be obtained
<a
href="http://salilab.org/imp/nightly/doc/html/installation.html#devbuild">from
our Subversion repository</a>.</p>
